# RimTalk Name Width Patch

修复 RimTalk 聊天窗口名字列宽度固定的问题。

原版 RimTalk 的名字列使用固定宽度，当对话字体大小变化时，名字容易换行导致显示不全。本补丁提供两种解决方式：

**模式一：限制最大宽度**
名字列宽度自适应，但不超过设定上限（默认 120px）。

**模式二：不限制宽度**
名字列完全自适应，不设上限，名字不换行显示。

可在游戏内模组设置中切换模式并调节宽度上限（40px ~ 800px）。
