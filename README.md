# RimTalk Name Width Patch

为 RimTalk 聊天窗口增加名字列宽度控制，提供两种模式：

**模式一：限制最大宽度**
超过设定宽度的名字被截断，对话区获得更多空间。默认 120px。

**模式二：不限制宽度**
保持 RimTalk 默认行为，名字不换行、宽度自适应。

可在游戏内模组设置中切换模式并调节宽度（40px ~ 800px）。
